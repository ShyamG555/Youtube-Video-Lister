# Youtube-Video-Lister

A simple web application that fetches and displays YouTube-style videos from the [FreeAPI YouTube endpoint](https://api.freeapi.app/api/v1/public/youtube/videos). Users can view video thumbnails, titles, and channel names, click to open videos on YouTube, and search for videos using a search bar.

## Features

- **Fetch and Display Videos**: Fetches a list of YouTube videos from the API and displays them in a grid layout.
- **Video Details**: Shows video thumbnails, titles, and channel names.
- **Open on YouTube**: Click on a video to open it in YouTube.
- **Search Bar**: Filter videos by title or channel name (frontend-only filtering).
- **Responsive Grid Layout**: Videos are displayed in a responsive grid for better UI.

---

## Technologies Used

- **HTML**: Structure of the web page.
- **CSS**: Styling and layout.
- **JavaScript**: Fetching data from the API, handling user interactions, and dynamic updates.

---

## How to Use

1. **View Videos**:
   - Videos are automatically fetched and displayed when the page loads.

2. **Search for Videos**:
   - Use the search bar at the top to filter videos by title or channel name.

3. **Open a Video**:
   - Click on a video card to open the video on YouTube in a new tab.

---

## Setup Instructions

### Prerequisites

- A modern web browser (e.g., Chrome, Firefox, Edge).
- An internet connection (to fetch videos from the API and open it on youtube).
